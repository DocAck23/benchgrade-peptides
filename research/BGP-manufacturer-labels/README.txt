BENCH GRADE PEPTIDES LLC — Per-SKU Vial Labels
================================================

Contents:
  107 vector label files (.svg), one per SKU, plus manifest.json.

File naming:
  Each filename is the SKU code (e.g. BGP-BPC-5.svg = BPC-157, 5mg vial).

Label sizes:
  3ml vial labels      — 450 × 225 px artboard
  10ml vial labels     — 675 × 375 px artboard
  10ml supply labels   — 675 × 375 px artboard (BAC water, acetic acid)

Print color:
  Maroon ground:  #5C1A1A
  Gold ink:       #B8923A
  Cream/white:    #F2EAD9

LOT / EXP on every label:
  LOT  H.012526
  EXP  01.2028

QR code:
  Each label has a dashed gold rectangle marking the QR-code print zone.
  Manufacturer prints the live LOT-specific QR (linking to the COA URL)
  inside that rectangle at fulfillment time.

Fonts (embedded via SVG <text>):
  Display  — Cormorant Garamond (700)
  Mono     — JetBrains Mono (500/600)
  Sans     — Inter (400/500/600)

Reference: manifest.json
  Lists every SKU with its compound name, dose, container size, and the
  filename of the corresponding label SVG.
